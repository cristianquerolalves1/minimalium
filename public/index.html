Installation

CDN

Add this line into your project‚Äôs <head>:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/cristianquerolalves/minimalium@latest/dist/style.css">

Local Installation

Clone or download the repository, build it, and include the CSS in your project:

# Clone the repository
git clone https://github.com/cristianquerolalves1/minimalium.git

# Navigate into the project folder
cd minimalium

# Build the project (Node.js required)
node build.js

# Include the generated CSS in your project
<link rel="stylesheet" href="./dist/style.css">

# Include Google Fonts (Inter recommended)
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800;900&display=swap" rel="stylesheet">
